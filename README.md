# Treino de IA com Python (PythonIA)

Projeto pessoal de treino de IA usando **Pandas** e **Scikit-learn**. Permite pré-processamento de dados, feature engineering e criação de modelos de linguagem, com avaliação de desempenho e ajuste de precisão.

## Estrutura

- `clientes.csv` e `novos_clientes.csv` → datasets  
- `inicial.ipynb` → notebook principal do treino  
- `Apostila Jornada Python.pdf` → material de referência  

## Como Rodar

1. Clone: `git clone https://github.com/sergiobfj/PythonIA.git`  
2. Instale dependências: `pip install pandas scikit-learn jupyter`  
3. Abra o notebook: `jupyter notebook inicial.ipynb` e execute as células.

## Objetivo

Demonstrar aplicação prática de LLMs em Python, explorando manipulação de dados, treino de modelos e avaliação de desempenho.
